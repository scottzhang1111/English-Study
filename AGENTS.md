# Eiken Vocab Lab

## Project Goal

This is a gamified English vocabulary learning app for children in Japan.

Goals:
- Help children memorize pronunciation
- Help children understand usage
- Make learning feel like a game
- Encourage daily study habits
- Use Pokemon as reward/collection system

## UI Style

- Soft pastel colors
- Rounded corners
- Friendly for elementary school children
- Minimal but playful
- Smooth animations
- Avoid overly complex layouts

## Important Rules

- Do NOT refactor the whole project unless explicitly asked
- Keep components small
- Prefer incremental changes
- Maintain current design language
- Avoid breaking existing features

## Learning Philosophy

The app should encourage:
- Active recall
- Repetition
- Listening
- Speaking
- Sentence usage

The app should NEVER punish children harshly.

Wrong answers should feel encouraging.

## Pokemon System

- Pokemon are rewards for studying
- Every child can own multiple Pokemon
- Studying gives EXP
- Max-level Pokemon unlock new Pokemon
- Use local cache for PokeAPI data

## Technical Rules

- Prefer TypeScript
- Prefer reusable components
- Keep API calls cached
- Avoid unnecessary dependencies
- Write tests for important logic
